Formwork Recycling


About

A nationwide first-of-its-kind platform designed to streamline the recycling and refurbishment of used formwork, including laminated wood, plank, and metal materials. This platform connects individuals and companies that own used formwork with specialized service providers who restore and repurpose these materials. By offering a centralized marketplace for buying, selling, and refurbishing formwork, the platform contributes to a cleaner environment and promotes the reuse of valuable construction materials.

Technologies & Contributions

• Developed the website framework using Angular
• Built the back-end with Django & Django REST Framework
• Implemented user authentication
• Deployed the application on AWS
• Established a version control workflow using GitHub
• Implemented Hotjar for monitoring user experience
• Designed and integrated a real-time notification system



The platform was developed in collaboration with DMX Construction, a company seeking an intuitive and efficient way for users to recycle and refurbish formwork. By simplifying the process of listing, browsing, and managing formwork materials, the platform ensures that construction resources are reused instead of discarded, reducing waste and promoting sustainability within the industry.

Designed with scalability and security in mind, the platform leverages Angular for a responsive user interface and Django REST Framework for robust back-end operations. The integration of real-time notifications, secure authentication, and AWS cloud deployment ensures a seamless experience for users. Through its intuitive design and optimized workflows, the platform enhances accessibility and encourages the adoption of sustainable construction practices.
